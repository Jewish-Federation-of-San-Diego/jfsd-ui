🔴 Cross-Cutting Issues (System-Wide)

Generic Titles (32/34 dashboards)

  • Every dashboard uses label titles ("Campaign Progress", "Spending Analysis") instead of insight headlines
  • Fix: "FY26 campaign hits 73% of goal — $4.2M raised, tracking 15% ahead of FY25" instead of "Campaign Progress"
  • This is the single highest-impact change across all dashboards
  • Only Share of Wallet and Major Gifts attempt insight-driven subtitles

Missing Annotations (30/34 dashboards)

  • Only Share of Wallet has proper quadrant annotations
  • No call-outs highlighting key insights, anomalies, or benchmarks
  • Fix: Add Plotly annotations/shapes for reference lines, top performers, outliers
  • Decision matrix requires: max 2 per chart, always annotate the main insight

Chart Crimes (8 dashboards)

  • Chart Gallery: Contains pie charts, 3D charts, gauges, polar charts — teaches bad patterns
  • Data Quality + Overview: Use circular progress = gauges
  • Major Gifts (potential): Funnel area chart could mislead
  • Fix priority: Gallery first (template source), then gauge replacements

Color Inconsistency

  • Some dashboards use semantic colors (SUCCESS/ERROR) ✅
  • Others use PLOTLY_COLORS decoratively ❌
  • Inconsistent application of teal=good, orange=bad rule
  • Fix: Audit every chart for semantic color mapping
